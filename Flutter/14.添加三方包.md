Flutter 的依赖管理在 `pubspec.yaml` 中。  

# 1.添加一个三方包

在 `pubspec.yaml` 的 **dependencies** 下添加依赖库：  

```
dependencies:
  flutter:
    sdk: flutter
  // 三方包
  english_words: ^3.1.3
```

添加三方包的格式很简单：`包名 + ^ + 版本号`。  

添加完成后，需要下载一下依赖。

# 2.其它添加三方包的方式

## 2.1 从本地添加

```
dependencies:
    // 包名
    pkg1:
        // 本地包路径
        path: ../../code/pkg1
```

## 2.2 从 Git 远程仓库添加

```
dependencies:
  // 包名
  pkg1:
    git:
      // 远程仓库 url
      url: git://github.com/flutter/packages.git
      // 远程仓库中的包的相对路径
      path: packages/package1
```
