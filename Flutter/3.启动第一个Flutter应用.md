# 2.Flutter 程序的入口

Flutter 的程序入口是一个 `main()` 函数：  

```
void main() => runApp( // Widget );
```

在 `main()` 函数中调用 `runApp()` 函数，传入一个 Widget。

Flutter 会将这个 Widget 视图添加到 **RenderView** 根节点上，这样 Widget 就能在视图树中被渲染出来。

这部分内容在本指南中就不深究了，简单了解一下就行，不影响上手开发。
