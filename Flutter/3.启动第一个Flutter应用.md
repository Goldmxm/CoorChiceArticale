# 2.Flutter 程序的入口

Flutter 的程序入口是一个 `main()` 函数：  

```
void main() => runApp( // Widget );
```

在 `main()` 函数中调用 `runApp()` 函数，传入一个 Widget，作为页面的根容器。  

构建一个 Flutter 页面，其实就是往这个根容器里添加各种 Widget。
